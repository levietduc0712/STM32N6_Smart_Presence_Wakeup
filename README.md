This project runs on the STM32N6570-DK with the B-CAMS-IMX MB1854 camera module using the Zephyr RTOS. The platform combines a Cortex-M55 MCU, onboard display, and a camera interface with a Time-of-Flight (ToF) sensor for depth sensing. 

It demonstrates a smart presence detection system that distinguishes between a person standing still and someone passing by. Only sustained presence triggers the display and live camera stream, enabling efficient and intelligent interaction for embedded vision applications.
